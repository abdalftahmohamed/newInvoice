# Invoicer (Laravel)

## Setup
1. `git clone https://github.com/abdalftahmohamed/newInvoice.git`
2. `cd newInvoice`
3. `composer install`
4. copy `.env.example` to `.env` and set DB credentials
5. `php artisan key:generate`
6. `php artisan migrate --seed`
7. `npm install && npm run dev`
8. `php artisan serve`
